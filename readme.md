## Instalação

- 1 Clonar o repositório

  - `git clone https://github.com/davidfaria/omnistack9-mobile-aircnc.git`

- 2 Instalar as dependências (Yarn ou NPM)

  - `npm install -g expo-cli`

  - `yarn install`

  - `npm install`

- 3 Start project (Yarn ou NPM)

  - `yarn run start`
  - `npm run start`
